# Repository for rorystock.com

## Tech Stack

- Built with [Nuxt.js](https://nuxtjs.org) / [Vue.js](https://vuejs.org)
- [Tailwind CSS](https://tailwindcss.com) for styling
- Deployed with Cloudflare Pages
